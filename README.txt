FERIENFIT AUF DEM IPHONE

1. Der Ordner enthält eine installierbare Web-App für die NRW-Sommerferien 2026.
2. Zum Testen am Computer: start-app.ps1 mit Rechtsklick „Mit PowerShell ausführen“.
3. Für das iPhone muss der Ordner über einen Webserver/Hosting-Dienst erreichbar sein (HTTPS).
4. Dann in Safari öffnen, Teilen antippen und „Zum Home-Bildschirm“ wählen.

Die Häkchen werden getrennt für jedes Kind direkt auf dem Gerät gespeichert.
Unter Einstellungen kann eine JSON-Sicherung exportiert und wieder eingelesen werden.
